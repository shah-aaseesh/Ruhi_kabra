

# Ruhi Kabra — Dark Maximalist Portfolio

## Overview
A dark-themed, animation-heavy portfolio website for a multidisciplinary filmmaker and artist. The design merges indie cinema aesthetics, Indian maximalism, and experimental art zine energy — all in a deeply immersive dark palette.

---

## Design System

- **Background:** `#0D0D0D` with subtle noise/grain texture overlay
- **Accent colors:** muted maroon (`#6B1D1D`), burnt orange (`#B85C2F`), deep indigo (`#2E1A47`), antique gold (`#C9A84C`)
- **Text:** warm off-white (`#EAE6DF`)
- **Textures:** CSS film grain overlay (animated), ink-bleed edges via gradients, subtle fabric-pattern backgrounds at low opacity
- **Typography:** Bold display font for headings (cinematic feel), clean readable body font
- **Custom cursor:** soft glowing orb that follows mouse

---

## Pages & Features

### 1. Home
- Fullscreen hero with large kinetic "RUHI KABRA" typography (animated letter-by-letter reveal)
- Subtitle: "Filmmaker • Artist • Writer" fading in beneath
- Slow-moving background textures (grain + subtle pattern drift)
- Floating, dimly lit elements that react to scroll (parallax layers)
- **About section** styled as a tilted, scanned handwritten note with soft spotlight glow
- **Featured films** strip — hover triggers glitch/grain flicker effect
- **Navigation blocks** to Film / Theatre / Art / Writing — asymmetric cards with hover glow

### 2. Film Page
- Header with contact info (name, location, phone, email)
- Dark background with animated grain overlay
- Asymmetrical video card grid (broken grid, overlapping)
- Each project: title, role, embedded YouTube video
- All 10+ projects listed with cinematic cards
- **Hover:** grain flicker + subtle zoom on cards
- **Click:** cinematic lightbox expand with fade-to-black transition
- "Have You Seen Tara" shows multiple episodes

### 3. Theatre Page
- Deep black background with moving spotlight gradient that follows cursor
- Content revealed with stage-lighting-inspired scroll animations
- 7 theatre projects listed with roles
- **Animations:** curtain-opening effect on page entry, text fades in sequentially like actors entering stage, spotlight follows cursor

### 4. Art Page
- Dark canvas background with layered textures
- Masonry grid layout (irregular, asymmetrical sizing)
- Placeholder art pieces displayed as scattered gallery items
- **Hover:** glow effect + zoom
- Slight floating parallax drift on scroll

### 5. Writing Page
- Dark editorial layout with soft off-white text
- Subtle paper texture behind text blocks
- Substack link and 3 articles listed (A Third Space, Doorway, My Name)
- **Hover:** underline grows slowly on links
- Text has subtle "breathing" shift animation

---

## Global Features

- **Film grain overlay** — persistent subtle animated noise across all pages
- **Smooth scroll** behavior site-wide
- **Page transitions** — fade-to-black between routes
- **Parallax scrolling** — foreground/midground/background layers
- **Custom glowing cursor** across the site
- **Responsive design** — adapts for mobile with simplified animations
- **Navigation** — minimal floating nav, blends into the dark aesthetic
